# indexer
A program to index text files and query words

I wrote the indexer in 2001 for a seminar paper for my state exam.

The source code is unchanged. The (inline) documentation is in German ;)

# Build
make

# Run
See "bin/runT.sh" for example how to index files.

Query of words from the index file is done with stdin/stdout.
